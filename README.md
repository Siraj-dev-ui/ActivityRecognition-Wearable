# Hardware Overview

This project uses the following hardware components:

- **Seeed XIAO nRF52840 Sense**  
  A compact microcontroller featuring a built-in IMU (accelerometer + gyroscope) and onboard LiPo charging support.

- **LiPo Battery (3.7 V, 500–800 mAh)**  
  Rechargeable lithium-polymer battery with a built-in protection circuit.

- **0.96″ OLED Display (SSD1306, I²C)**  
  Used to display real-time activity classification results.
